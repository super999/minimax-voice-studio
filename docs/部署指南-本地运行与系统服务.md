# Minimax Voice Studio 本地部署指南

本文档说明如何在本地运行 Minimax Voice Studio 并配置为 Windows 系统服务。

## 快速启动

```bash
# 进入项目目录
cd D:\claude-code-work\minimax-voice-studio\.worktrees\impl-minimax-voice

# 启动应用（端口 8601，监听 0.0.0.0）
PORT=8601 pm2 start npm --name "minimax-voice-studio" -- start
```

## 应用信息

| 项目 | 值 |
|------|-----|
| 访问地址 | http://localhost:8601 |
| 监听地址 | 0.0.0.0:8601 |
| 进程管理器 | PM2 |
| 技术栈 | Next.js 14 |

## PM2 进程管理命令

```bash
# 查看所有进程状态
pm2 status

# 查看应用日志
pm2 logs minimax-voice-studio

# 重启应用
pm2 restart minimax-voice-studio

# 停止应用
pm2 stop minimax-voice-studio

# 删除进程
pm2 delete minimax-voice-studio

# 保存当前进程列表（用于开机恢复）
pm2 save

# 恢复已保存的进程列表
pm2 resurrect
```

## Windows 系统服务配置

### 方式一：任务计划程序（推荐）

以**管理员身份**打开 PowerShell，执行以下命令：

```powershell
# 创建开机启动任务
$action = New-ScheduledTaskAction -Execute 'C:\nvm4w\nodejs\node.exe' -Argument 'node_modules/next/dist/bin/next start -p 8601' -WorkingDirectory 'D:\claude-code-work\minimax-voice-studio\.worktrees\impl-minimax-voice'
$trigger = New-ScheduledTaskTrigger -AtStartup
$settings = New-ScheduledTaskSettingsSet -AllowStartIfOnBatteries -DontStopIfGoingOnBatteries -StartWhenAvailable
Register-ScheduledTask -TaskName 'MinimaxVoiceStudio' -Action $action -Trigger $trigger -Settings $settings -Description 'Minimax Voice Studio - Next.js Application'

# 立即启动任务
Start-ScheduledTask -TaskName 'MinimaxVoiceStudio'

# 查看任务状态
Get-ScheduledTask -TaskName 'MinimaxVoiceStudio' | Get-ScheduledTaskInfo

# 删除任务（如需卸载）
Unregister-ScheduledTask -TaskName 'MinimaxVoiceStudio' -Confirm:$false
```

### 方式二：PM2 自启动

```bash
# 生成启动命令（需要管理员权限）
pm2 startup

# 保存当前进程状态
pm2 save
```

## 常见问题

### Q: 端口被占用
```bash
# 查看 8601 端口占用情况
netstat -ano | findstr 8601

# 结束占用进程（替换 <PID> 为实际进程ID）
taskkill /PID <PID> /F
```

### Q: 服务无法启动
```bash
# 查看详细日志
pm2 logs minimax-voice-studio --err --lines 50

# 检查 .env 文件是否存在
test -f .env && echo "存在" || echo "不存在"
```

### Q: 需要修改端口
修改启动命令中的 `-p 8601` 为其他端口号。

## 环境变量

确保 `.env` 文件包含以下配置：

```env
DATABASE_URL="mysql://user:password@host:3306/database"
MINIMAX_API_KEY="your-api-key-here"
MINIMAX_API_HOST="https://api.minimaxi.com"
NEXTAUTH_SECRET="your-secret-key-change-in-production"
NEXTAUTH_URL="http://localhost:8601"
```
